# IvueIcon 图标字体


图标可以使用图标字体或外部SVG。此外，可以接受一些大小尺寸，如 ```2x, 3x, 4x and 5x``` . 默认情况下, IvueIcon 使用的是 [material io](https://material.io/tools/icons/?icon=mood_bad&style=baseline) 图标,您必须使用下划线而不是空格来传递图标的名称。例如：```<IvueIcon>thumb_up</IvueIcon>```
